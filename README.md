SETUP (2 minutes)

Before deployment:
- Everything works locally by default
- Forms show validation but don't send (expected)

After deployment to Netlify:
1. Go to analytics.google.com - Create property - Get tracking ID
2. Replace G-PLACEHOLDER in index.html with your tracking ID
3. Push to GitHub and redeploy
4. Forms automatically work

CUSTOMIZE
